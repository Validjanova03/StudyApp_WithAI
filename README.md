# StudyWAI

StudyWAI is a desktop application designed to help students organize and review programming study notes. The app provides a simple and modern interface for saving lessons, highlighting Python code, and managing study schedules.

## Features

- Save and manage study notes
- Python syntax highlighting for code snippets
- Study reminders based on selected dates
- Mark difficult topics for later review
- Modern dark-mode interface
- AI-powered note summaries (optional)

## Technologies Used

- Python
- CustomTkinter (GUI)
- SQLite (local database)
- Threading (reminder system)
- Regular expressions (syntax highlighting)
